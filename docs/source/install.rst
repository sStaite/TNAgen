Installation
============

To use TNAgen, first install it using your preferred method:

Pip:
.. code-block:: console

   $ pip install TNAgen


Conda:
.. code-block:: console

   $ conda install TNAgen


From source: 
.. code-block:: console

   $ git clone https://github.com/sStaite/TNAgen
   $ cd TNAgen
   $ python setup.py install
