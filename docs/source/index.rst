Welcome to TNAgen's documentation!
==================================

Introduction
============
``TNAgen`` is a python package that allows users to generate transient noise artifacts (glitches) in gravitational-wave detector data using a GAN model. 

Limitations
-----------
At the moment, there are only a few functions up and running, but the package is in early development stages.
